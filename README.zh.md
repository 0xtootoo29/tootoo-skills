# Skill Router

[English](./README.md) | 简体中文

智能 Skill 路由器 - 通过交互式问答帮助你找到最合适的 Skill

## 为什么需要 Skill Router？

当你安装了大量 Claude Code Skills 后，会遇到这些问题：

- 🤔 记不清每个 skill 的具体名称
- 🔀 同类型的 skills 太多，不知道该用哪个
- 📝 想"写文章"，但有多个写作相关的 skills
- 🎨 想"生图"，但有封面、漫画、配图等多种选择
- 🚀 不想记住斜杠命令，只想说"我要做什么"

**Skill Router 就是为了解决这些问题而生的。**

## 使用方法

安装后，直接用自然语言描述你想做什么：

```
"生图" / "画图"
"写文章" / "写作"
"发布内容" / "分发内容"
"帮我选个 skill"
```

Skill Router 会：
1. 理解你的意图
2. 通过可点击的选项卡询问细节
3. 推荐最合适的 skill
4. 展示相关平台规则（如适用）

## 核心功能

### 🔄 自动发现
**每次运行时自动扫描 `~/.claude/skills/` 目录**。新安装的 skills 立即可用，无需配置。

### 🎯 智能分类
分析每个 skill 的描述和触发词，自动归类：
- 图像生成（海报、漫画、配图）
- 内容创作（写作、编辑、提取）
- 内容分发（X、小红书、公众号、YouTube）
- 开发工具（API、前端、数据库）
- 知识管理（调研、下载、转换）
- AI 工具管理（Skill、MCP、CLAUDE.md）

### 📋 交互式选项
使用可点击的选项卡而不是文本输入 - 更快更直观。

### 📐 平台规则
在相关时展示平台特定要求：
- **X (Twitter)**：16:9 或 1:1，<5MB
- **小红书**：3:4 竖图，<10MB
- **公众号**：16:9，<2MB
- **YouTube**：16:9，<2MB

### 🧠 学习偏好
记住你的选择和自定义触发词，优化未来推荐。

### 🎨 用户友好
- 交互过程中隐藏技术细节
- 仅在最终推荐时显示 skill 技术名称
- 渐进式提问 - 每次只问一个关键问题
- 使用通俗语言描述

## 安装

```bash
# 从分支克隆
git clone -b skill-router https://github.com/0xtootoo29/tootoo-skills.git skill-router
cp -r skill-router ~/.claude/skills/

# 或从主分支安装
git clone https://github.com/0xtootoo29/tootoo-skills.git
cp -r tootoo-skills/skill-router ~/.claude/skills/
```

无需配置 - 开箱即用。

## 工作原理

1. **自动扫描**：扫描 `~/.claude/skills/` 发现所有已安装的 skills
2. **意图识别**：分析你的输入判断类别
3. **渐进式问答**：通过选项卡询问细节
4. **智能推荐**：推荐最佳 skill，显示通俗描述 + 技术名称
5. **偏好学习**：保存你的选择，优化未来推荐

## 适用人群

- 🆕 **新手**：不熟悉 skills，需要引导
- 🚀 **效率党**：不想记住所有 skill 名称
- 🎯 **选择困难症**：同类 skills 太多，不知道选哪个
- 🧠 **自然交互爱好者**：喜欢说"我要做什么"而不是记命令
